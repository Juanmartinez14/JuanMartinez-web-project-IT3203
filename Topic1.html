Early Development of HTTP

The Origin of HTTP

HTTP was made in the late 1980s by an inventor named Tim Berners-Lee at a company named CERN. The main purpose of the HTTP was to help scientists share information more effecient.

HTTP/0.9

The first version of HTTP was the HTTP/0.9 and it was super simple. The HTTP could only use the GET method and also had the abilty to retrieve plain text documents. It also had no headers or status codes.
